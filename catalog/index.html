Веб-приложение

Каталог игр

Каталог представляет собой легковесное, быстродействующее одностраничное веб-приложение с удобной навигацией по платформам, быстрым поиском и другими возможностями. Среди них - вишлист, детский фильтр, возможность бронирования. Подробности ниже.

Адаптивный дизайн

Сетка приложения подстраивается под размеры разных устройств, выглядит и работает одинаково хорошо, будь то смартфон, планшет, ноутбук или монитор настольного компьютера.

Быстрый переход
на страницу игры

Переход с общего списка игр на страницу конкретной игры - без загрузок и ожиданий - так, как и должно работать одностраничное веб-приложение.

Вишлист

Помимо вкладок для разных платформ предусмотрена отдельная вкладка для списка избранных игр. Избранные игры сохраняются в памяти устройства и никуда не пропадают при закрытии браузера.

Реактивный поиск

Поиск игры по названию - начинайте вводить запрос и результаты мгновенно отобразятся на странице.

Детский фильтр

Чтобы временно исключить игры, не предназначенные для детской аудитории - отметьте соответствующий флажок.

Бронирование мест

Предоставьте своим клиентам возможность заблаговременно забронировать место. После отправки заполненной формы на e-mail приходит уведомление с указанными данными.
